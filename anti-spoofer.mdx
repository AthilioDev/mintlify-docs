# Athilio Shield Protection

Athilio Shield Protection é o sistema integrado de licenciamento seguro e proteção avançada do Athilio Spotify. Ele combina validação de licença, autenticação por IP e gerenciamento de franquia de servidores.

## Como funciona

- Gera identificadores únicos a partir de informações de hardware, sessão e infraestrutura do servidor.
- Detecta tentativas de evasão e reutilização indevida de licenças.
- Aplica bloqueios automáticos em perfis comprometidos.
- Permite revogação dinâmica e gestão de bloqueios pelo painel em nuvem.

## Arquitetura do sistema

- Cliente FiveM → resource `Athilio_Spotify`
- Resource Athilio → API de validação centralizada
- API de validação → Dashboard em nuvem
- Dashboard → gerenciamento de licenças, IP e franquias de servidores

## Validação de licença com autenticação por IP

- O sistema valida o uso da licença junto ao IP autorizado no painel.
- Servidores não autorizados são bloqueados automaticamente.
- Mudança de VPS ou alteração de IP exige redefinição no painel.
- O sistema monitora ativações simultâneas não autorizadas.

## Franquia de servidores e proteção distribuída

- O proprietário pode vincular uma franquia de servidores ao mesmo ecossistema.
- Os servidores vinculados podem receber sincronização automática de bloqueios e validações.
- Tentativas de retorno utilizando spoofing ou alteração de identificadores são automaticamente bloqueadas.
- O painel sincroniza uma blacklist distribuída entre servidores vinculados.

### Tipos de bloqueio detectados

- HWID
- Steam ID
- Discord ID
- whitelist (WL)
- IP
- identificadores de sessão e hardware

<Warning>
  Se o usuário tentar alterar HWID, Steam, Discord, WL ou IP para voltar, o acesso ao ecossistema Athilio é automaticamente bloqueado.
</Warning>

## Integração com painel administrativo

- O painel administra licenças, IPs autorizados e bloqueios.
- Exibe status de validação e histórico de ativações.
- Permite revogar ou liberar acessos em tempo real.

<Note>
  O painel centralizado permite que o dono do servidor gerencie toda a franquia de servidores a partir de um local único.
</Note>

## Recursos

- Sistema standalone de licenciamento
- Dashboard cloud para gerenciamento
- Autenticação por IP
- Proteção contra spoofing e reutilização de licença
- Controle de franquias de servidores
- Blacklist distribuída sincronizada
- Revogação remota de bloqueios

## Versões

- A proteção integrada está disponível para versões acima de `1.0.2`.
- Versões mais antigas podem não possuir todas as funcionalidades do painel.

<Note>
  Athilio Shield Protection funciona em conjunto com a licença do Athilio Spotify e é a camada de proteção recomendada para uso profissional.
</Note>
