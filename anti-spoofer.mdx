# Anti-Spoofer

O Anti-Spoofer da Athilio Software é uma proteção integrada que funciona diretamente na infraestrutura do resource.

## Como funciona

- Gera uma assinatura criptográfica de rede baseada em hardware e blocos lógicos.
- Detecta perfis nocivos em nível de infraestrutura.
- Aplica banimento permanente para perfis comprometidos.
- Permite revogações dinâmicas via nuvem.

## Banimento por IP e proteção de servidor

- O sistema valida o IP autorizado no painel.
- Qualquer servidor não autorizado é bloqueado imediatamente.
- Mudança de VPS ou IP exige redefinição no painel.
- Licenças usadas em múltiplos servidores são banidas.

## Proteção avançada e franquia de servidores

Se um servidor que usa o script for banido, o dono pode vincular uma franquia de servidores.

Isso significa que:

- banimentos em um servidor podem se estender a toda a franquia
- tentativas de spoofing com HWID, Steam, Discord, whitelist ou IP são detectadas
- o sistema banirá o perfil em todos os servidores vinculados
- o usuário não consegue retornar por meio desse script

### Tipos de ban detectados

- HWID
- Steam ID
- Discord ID
- whitelist (WL)
- IP
- assinaturas de rede e hardware

<Warning>
  Se o usuário tentar mudar HWID, Steam, Discord, WL ou IP para voltar, o sistema bloqueia o acesso por esse script.
</Warning>

## Integração com painel admin

- O painel administra licenças, IPs autorizados e bloqueios.
- Ele exibe status de validação e histórico de ativações.
- A interface permite liberar ou revogar assinaturas diretamente na nuvem.

## Versões

- A proteção integrada está disponível para versões acima de `1.0.2`.
- Versões mais antigas podem não possuir todas as funcionalidades do painel.
