# Athilio Software

Bem-vindo à documentação oficial do `Athilio_Spotify`.

Athilio_Spotify é um script standalone de música para servidores FiveM. Este site reúne todas as informações de instalação, licença, dashboard, proteção por IP e suporte.

<Info>
  A barra contextual está configurada para copiar conteúdo, abrir em assistentes de IA e adicionar este docs como um servidor MCP.
</Info>

## O que você encontra aqui

- Instalação passo a passo do resource
- Configuração da licença e validação
- Uso do painel de Dashboard
- Proteção por IP e bloqueio de licença
- Suporte oficial e contatos da equipe
