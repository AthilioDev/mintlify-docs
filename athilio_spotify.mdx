# Configuração do Athilio Spotify

`athilio_spotify` é o resource de música para FiveM com gerenciamento via Dashboard.

> Instalação e passos de deploy foram movidos para a página **Instalação**. Consulte [Instalação](instalacao.mdx) para instruções passo a passo.

## Licença

O arquivo `athilio_spotify/license.json` deve conter a licença ativa.

### Exemplo

```json
{
  "license": "ATHILIO-XXXX-XXXX-XXXX"
}
```

### Passos rápidos

1. Acesse o Dashboard Athilio e copie a licença ativa.
2. Cole em `athilio_spotify/license.json` no campo `license`.
3. Reinicie o servidor.

<Warning>
  Não compartilhe sua licença publicamente. Uso indevido em vários servidores pode resultar em bloqueio.
</Warning>

## Configuração por versão

### Versão 1.0.2

Opções principais são configuradas em `config.lua`.

```lua
Config = {}
Config.License = ""
Config.ServerName = "AthilioFy"
Config.HexColor = "27ae60"
Config.Logo = ""
Config.YoutubeAPIKey = ""
Config.Command = ""
Config.DefaultVolume = 50
Config.PropModel = ""
Config.PropFallback = ""
Config.EnableAnimation = true
Config.AnimDict = ""
Config.AnimName = ""
Config.PropBone = 57005
Config.PropPlacement = {0.26, -0.12, -0.10, 80.0, -160.0, 180.0}
```


### Versões acima de 1.0.2

Configurações são gerenciadas via Dashboard.

## Dashboard

O Dashboard gerencia:

- licença e IP autorizado
- configurações do resource
- histórico de validações
- bloqueios e revogações

<Warning>
  Se o painel não estiver disponível, use `config.lua` para configuração local.
</Warning>

## Validação e problemas comuns

- Verifique `license.json`, `server.cfg` e dependências.
- Limpe o cache do FiveM após alterações significativas.
