# Athilio Shield Protection

Athilio Shield Protection é a camada de proteção que complementa o resource `athilio_spotify` com regras de validação adicionais.

## Como funciona

- Coleta identificadores de sessão, IP e hardware quando o resource tenta validar.
- Compara os dados com o registro autorizado no painel.
- Detecta alterações comuns de identificadores e aplica ações de bloqueio ou alerta configuráveis.
- Permite revogação e controle de acesso a partir do painel.

## Arquitetura do sistema

- Cliente FiveM → resource `athilio_spotify`
- `athilio_spotify` → API de validação centralizada
- API de validação → Dashboard Cloud
- Dashboard Cloud → gerenciamento de licença, IP e bloqueios

## Validação de licença e IP

- O resource valida a licença com base no registro do painel.
- O IP autorizado deve corresponder ao servidor em produção.
- Mudança de VPS ou alteração de IP exige atualização no painel.
- O sistema monitora tentativas de uso simultâneo em múltiplos servidores.

## Proteção de franquias

- O proprietário pode vincular servidores a uma franquia administrada.
- Servidores vinculados compartilham regras de bloqueio e status de validação.
- A blacklist distribuída auxilia na propagação de bloqueios entre servidores autorizados.
- Alterações de identificadores não autorizadas podem ser bloqueadas de acordo com regras configuradas.

## Limitações

- A proteção baseia-se em regras de licença e identificadores, não em detecção antifraude perfeita.
- Mudanças legítimas de hardware ou IP devem ser atualizadas no painel.
- A proteção ajuda a reduzir uso indevido, mas não substitui auditoria de servidor.

## Camadas de proteção

- Validação de licença
- Autenticação por IP
- Verificação de sessão
- Identificadores de hardware e de sessão
- Blacklist distribuída
- Sincronização entre franquias

## Integração com o painel

- O painel administra licenças, IPs autorizados e bloqueios.
- Exibe status de validação e histórico de ativações.
- Permite revogar ou liberar acessos em tempo real.

<Note>
  O painel centralizado permite gerenciar toda a franquia de servidores de um único local.
</Note>

## Recursos

- Camada de proteção para `athilio_spotify`
- Dashboard Cloud para gerenciamento
- Autenticação por IP
- Detecção de alterações comuns de identificadores
- Controle de franquias de servidores
- Blacklist distribuída sincronizada
- Revogação remota de acessos

- Versões mais antigas podem não ter todas as funcionalidades do painel.
