# FAQ

## Posso trocar de VPS sem perder a licença?

Sim. Você deve redefinir o IP autorizado no painel do Athilio antes de iniciar o resource no novo servidor.

## Como redefinir o IP autorizado?

Acesse o Dashboard Cloud e atualize o servidor autorizado. Em seguida, reinicie o resource.

## Posso usar em um servidor de desenvolvimento?

Sim, desde que o IP do servidor de desenvolvimento esteja autorizado no painel e a licença não seja usada simultaneamente em outro servidor.

## O que acontece se a licença for compartilhada?

O sistema detecta ativações simultâneas não autorizadas e pode bloquear o acesso ao ecossistema Athilio automaticamente.

## O Dashboard funciona em versões antigas?

O Dashboard Cloud está disponível em versões acima de `1.0.2`. Em versões antigas, use `config.lua` para configurar o resource.

## Existe suporte para bases customizadas?

Sim. Athilio Spotify é standalone e compatível com bases customizadas, Creative, vRP, QBCore, ESX e variantes híbridas.

## Como funciona a revogação de banimentos?

O dono do servidor pode liberar ou revogar acessos pelo painel administrativo. As mudanças são aplicadas em tempo real.
